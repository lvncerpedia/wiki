# OS

OS（Operating System / オペレーティングシステム）とは、パソコンやスマートフォンなどのデバイス全体を管理し、アプリを動かすための「基本ソフトウェア」です。
人間と機械（ハードウェア）をつなぐ橋渡しのような役割を果たします。

## 主な役割

OSがないデバイスは、ただの「箱」になってしまいます。

- ハードウェアの管理: CPUやメモリ、キーボード、マウスなどの部品を動かし、制御します。
- アプリの実行環境: アプリがスムーズに動くための土台を提供します。
- ユーザーインターフェース（UI）の提供: 画面のタップやマウス操作など、人間が直感的に操作できる環境を作ります。

## 代表的なOSの種類

デバイスや用途によって、さまざまなOSが使われています。

### パソコン向け

- Windows: 世界中で最も普及しているOS。
- macOS: Apple社のMacシリーズに搭載されているOS。

### スマートフォン・タブレット向け

- iOS: iPhoneやiPad専用のOS。
- Android: Googleが開発し、多くのメーカーのスマホに搭載されているOS。

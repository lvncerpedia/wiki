# クラウド・インフラ系イベント

## 日本（総合・マルチクラウド）

| イベント | 時期 | 地域 | 特徴 |
| --- | --- | --- | --- |
| [AWS Summit Japan](https://aws.amazon.com/jp/events/summits/) | 春〜夏 | 東京など | 超大型。AWS 最新サービス・事例が一気に集まる |
| [NexTech Week](https://nextech-week.jp/) | 春・秋 | 東京 / 幕張 | AI・クラウド・ブロックチェーンなど横断型。企業の導入検討向き |
| [Developers Summit](https://event.shoeisha.jp/devsumi) | 春・秋 | 東京 | 日本最大級の開発者向けカンファレンス。クラウド・インフラ枠も厚い |
| [Infrastructure Engineers Meetup](https://infra-engineer.connpass.com/) | 月1回程度 | 東京中心 | インフラエンジニアの定番ミートアップ |

## Kubernetes / コンテナ

| イベント | 時期 | 地域 | 特徴 |
| --- | --- | --- | --- |
| [CloudNative Days](https://cloudnativedays.jp/) | 年複数回 | 日本各地 | Kubernetes 文化圏の中心。CNDT が最大級 |
| [KubeCon + CloudNativeCon](https://events.linuxfoundation.org/kubecon-cloudnativecon-north-america/) | 年複数回 | 世界各地 | CNCF 旗艦。世界最大級の Kubernetes イベント |
| [Container Days](https://containerdays.io/) | 年1回程度 | 日本 | コンテナ・K8s 実践寄り |
| DockerCon | 不定期 | 海外 | Docker 公式（近年はオンライン中心） |

## AWS

| イベント | 時期 | 地域 | 特徴 |
| --- | --- | --- | --- |
| [JAWS DAYS](https://jawsdays.jp/) | 毎年9月ごろ | 日本 | 日本 AWS ユーザーグループ最大級 |
| [AWS Summit Japan](https://aws.amazon.com/jp/events/summits/) | 春〜夏 | 日本 | 国内向け大型サミット |
| [AWS re:Invent](https://reinvent.awsevents.com/) | 毎年11〜12月 | ラスベガス | AWS 最大イベント。新発表の本丸 |
| [AWS re:Inforce](https://reinforce.awsevents.com/) | 毎年6月ごろ | 米国 | セキュリティ特化 |
| JAWS-UG 支部ミートアップ | 年間通して | 日本各地 | 地域ごとの小規模勉強会。気軽に参加しやすい |

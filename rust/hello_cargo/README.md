# Hello, Cargo

## Whats this??

Cargo は Rust のビルドシステム兼パッケージマネージャである。

- Cargo がインストールされているか確認

  ```rust
  cargo --version
  ```

## Commands

| commands                | description                                                            |
| ----------------------- | ---------------------------------------------------------------------- |
| `cargo new`             | プロジェクトを作成できる                                               |
| `cargo build`           | プロジェクトをビルドできる                                             |
| `cargo run`             | プロジェクトのビルドと実行を 1 ステップで行える                        |
| `cargo check`           | バイナリを生成せずにプロジェクトをビルドして、エラーがないか確認できる |
| `cargo build --release` | 最終的にリリースできるようになったら、最適化した状態でコンパイルできる |

# ローグライク vs ローグライト

## 核心の違い

| 要素 | ローグライク | ローグライト |
|------|------------|------------|
| 元ネタ | Rogue(1980)の忠実な再現 | ローグ風の「雰囲気だけ」借りたもの |
| 死んだら | 完全リセット | 引き継ぎ要素あり |
| 成長 | キャラは毎回ゼロから | ランの外で永続強化できる |
| 難易度 | 高め・ストイック | 間口が広い |
| ターン制 | 基本ターン制 | アクションでもOK |

## 具体的なゲームで見ると

### ローグライク寄り**

- **Nethack / DCSS** → 死んだら全部消える、引き継ぎ一切なし
- **Caves of Qud** → 死=即終了、世界生成もランダム
- **Elona** → 自由度化け物級、でも死は重い

### ローグライト寄り

- **Hades** → 死ぬたびにネクター貯まってキャラ強化できる
- **Slay the Spire** → ランの外での解放要素あり
- **Dead Cells** → 死んでも細胞（通貨）で永続アップグレード

## エピソード的な話

### Hades のうまさ

「死ぬ」こと自体がストーリーに組み込まれてて、死ぬたびに父・ハデスとの会話が進む設計。
「死=ゲームオーバー」じゃなくて「死=進行」になってる。
ローグライトの引き継ぎ要素を**ナラティブで正当化**した天才例。

### Nethackのカオスさ

1987年から開発が続いてて、未だに「犬に食われる」「自分の爆発に巻き込まれる」みたいな死に方が現役。
コミュニティが「YASD（Yet Another Stupid Death＝またバカな死に方した）」って言葉を作ったくらい死が文化になってる笑

ざっくりいうと、
**ローグライク＝ストイックな死に覚えゲー**
**ローグライト＝死んでも少しずつ前進できる設計**

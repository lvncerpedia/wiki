## 🕹️ 第1章：FPSの夜明け（1990〜1996）

### Wolfenstein 3D (1992) - id Software
- 当時としては革命的な「主観視点」
- 実際には2.5D（壁が全部同じ高さ）
- ナチスの城から脱出するゲーム
- 暴力表現が問題視され→後の**レーティング制度創設**につながる

### DOOM (1993) - id Software
```
DOOM が生んだもの
├─ MOD文化（ユーザー改造）
├─ ネットワーク対戦（LAN）
├─ 「Doomクローン」という言葉が生まれるほど模倣作が乱立
└─ 学校や職場でのプレイが社会問題に（笑）
```
- 販売形式も革新的：**シェアウェア**（最初の章は無料配布）

### Quake (1996) - id Software
- **完全なポリゴン3D**、垂直方向の視点移動が可能に
- インターネット対戦が本格化
- **ロケットジャンプ**など高度な動きが生まれる
- e-Sportsの原点：**QuakeCon**大会が始まる

---

## 🌐 第2章：オンライン対戦の確立（1997〜2003）

### GoldenEye 007 (1997) - Rare / N64
- コンソールでFPSが遊べる！の証明
- **4人分割画面**マルチプレイが大流行
- エイムアシストの原型もここから

### Half-Life (1998) - Valve
```
Half-Lifeの革新
├─ ムービーなしでストーリーを語る「没入型演出」
├─ MODとして Counter-Strike が生まれる
└─ ゲームエンジンのライセンス商売モデルを確立
```

### Counter-Strike (2000) - 元はHalf-LifeのMOD！
- テロリスト vs 対テロ部隊
- **1発の重み、お金管理、チーム連携**
- 今でも CS2 として現役←すごくない？

### Halo: Combat Evolved (2001) - Bungie / Xbox
- **体力自動回復**（それまでは回復アイテム拾い）
- 武器2つ持ち制限で戦略性アップ
- コンソールFPSの操作性を完成させた

---

## 💥 第3章：大衆化とミリタリー全盛（2003〜2011）

### Call of Duty (2003〜)
```
CoD進化の流れ
│
├─ CoD1〜2：WW2、映画的演出が売り
├─ CoD4 Modern Warfare (2007)：★転換点★
│   ├─ 現代戦争テーマへ
│   ├─ キルストリーク報酬
│   ├─ XP・ランク・アンロック
│   └─ → 今のFPS全部これ影響受けてる
└─ MW2 (2009)：シリーズ最大ヒット
```

### Battlefield シリーズ (2002〜) - DICE
- CoD と対比される存在
- **大人数・乗り物・広大なマップ**
- 「BFかCoDか」論争が2010年代まで続く

### Team Fortress 2 (2007) - Valve
- キャラクターに個性を持たせたヒーロー系の先祖
- **基本無料化**（2011）でF2Pモデルを広める

---

## 🚀 第4章：多様化と新ジャンル誕生（2012〜2019）

### Titanfall / Apex の文脈
- **壁走り・高機動アクション**の流行

### Overwatch (2016) - Blizzard　★大転換点★
- ヒーロースキル＋FPS
- キャラ格差・メタゲームの概念が普及
- eスポーツリーグ（OWL）設立

### Battle Royale の爆発
```
BR旋風の流れ
│
├─ 2017 PUBG：リアル系・緊張感
├─ 2017 Fortnite：建築＋BR、基本無料、10代に大普及
└─ 2019 Apex Legends：
        ├─ ピン（マーカー）システム
        ├─ 復活システム
        └─ → チーム連携の新スタンダードに
```

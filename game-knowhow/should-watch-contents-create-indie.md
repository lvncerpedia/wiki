# まず最優先で見てほしいやつ

## YouTube / 講演

### GDC (Game Developers Conference) YouTube

https://www.youtube.com/@Gdconf

ゲーム開発者の大学みたいな場所。

特におすすめ：

- postmortem（開発振り返り）
- level design
- combat design
- narrative design
- indie marketing

インディー作るなら「成功談」より「失敗談」が超参考になる。

### GMTK (Game Maker’s Toolkit)

https://www.youtube.com/@GMTK

ゲームデザイン分析系で最強クラス。

特に：

- 導線
- レベルデザイン
- UX
- “なぜ気持ちいいか”

を言語化してくれる。
あと毎年のGMTK Game Jamも超有名。

### Masahiro Sakurai on Creating Games

https://www.youtube.com/@sora_sakurai_en

桜井政博 のチャンネル。
「ゲームって結局どう作れば気持ちいいの？」が詰まってる。

インディーにも直結する：

- 情報量
- テンポ
- エフェクト
- プレイヤー心理

### Adam Millard - The Architect of Games

https://www.youtube.com/@ArchitectofGames

「ゲーム体験」を分析するタイプ。
独特なゲーム作りたい人に刺さる。

### AI and Games

https://www.youtube.com/@AIandGames

ゲームAI特化。

敵AIだけじゃなく：

- ディレクション
- ストーリー
- 生成システム
  まで扱う。

### Design Doc

https://www.youtube.com/@DesignDoc

ゲームデザインの細かい話が多い。
ローグライクとか作るならかなり良い。

## イベント（超重要）

### 世界最大級

## かなり重要な視点

インディー開発って、「技術力」だけで勝つ世界じゃなくて、

- 変な体験
- 独特な空気
- 誰かの記憶に残る瞬間
- スクショ1枚で気になる見た目

が超強い。
だから、ゲームだけじゃなくて、

- 映画
- 建築
- 音楽
- 現代アート
- 都市
- 廃墟
- ネット文化

まで吸収するとかなり化ける。
特に君みたいに「独特な世界観」方向に興味あるタイプは、“ゲームだけ見てる”状態になるともったいない。

# SUCCESsの法則

スタンフォード大学のチップ・ハース教授とダン・ハース兄弟が著書『アイデアのちから』で提唱した、「人の記憶に残り、行動を促すメッセージ」に共通する6つの要素です。
頭文字をとって「SUCCESs」と呼ばれ、マーケティングやプレゼンテーションで広く活用されています。

## 6つの要素（SUCCESs）

| 要素 | 説明 |
| - | - |
| S (Simple：単純明快) | メッセージの核心を突くこと。情報を極限まで削ぎ落とし、最も重要な1点に絞る。 |
| U (Unexpected：意外性) | 聞き手の常識や予想を裏切り、注意を惹きつけること。あえて「欠落」を作って好奇心を刺激する。 |
| C (Concrete：具体性) | 誰が聞いても同じイメージを思い浮かべられるようにすること。五感（見る・聞く・触るなど）で感じられる表現を使う。 |
| C (Credible：信頼性) | 信じてもらうための根拠を示すこと。専門家の意見や権威、具体的な数字、あるいは受け手が自ら検証できる要素を取り入れる。 |
| E (Emotional：感情に訴える) | 聞き手の心（エモーション）に響かせること。論理だけでなく、「自分に関係がある」「大切だ」と感情を動かす。 |
| S (Story：物語性) | 単なる事実の羅列ではなく、ストーリー（物語）として伝えること。聞き手の想像力をかき立て、行動へのモチベーションを生む。 |

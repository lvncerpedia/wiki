## 機能別組織(Functional Organization)

営業・開発・製造みたいに「機能」ごとに部門を分ける形。

```
              社長
        ┌──────┼──────┐
      営業部   開発部   製造部
```

専門性がガッと上がるし、同じ機能の人が集まるから知識の蓄積もしやすい。
でも部門間の壁ができやすくて、「営業は売る、開発は作る」で終わっちゃって事業全体の視点が薄くなりがちなのが弱点。

## 事業部制組織(Divisional Organization)

製品ラインや地域ごとに事業部を作って、その中に営業・開発・製造を全部持たせる形。ミニ会社が複数あるイメージ。

```
                社長
        ┌─────────┼─────────┐
      A事業部     B事業部     C事業部
     ┌──┼──┐    ┌──┼──┐    ┌──┼──┐
   営業 開発 製造  営業 開発 製造  営業 開発 製造
```

事業ごとに採算がはっきりするし、市場の変化にも事業部単位で素早く反応できる。ただ機能が事業部ごとに重複するから、コストはかさみがちなんだよね。

## 比較表

| 観点 | 機能別組織 | 事業部制組織 |
|---|---|---|
| 専門性 | 高い | 機能別より分散しがち |
| 意思決定スピード | 遅め(全社調整が必要) | 速い(事業部内で完結) |
| 部門間連携 | 弱い(サイロ化しやすい) | 同事業部内は強い |
| 採算の見える化 | 不明瞭 | 事業部ごとに明確 |
| コスト効率 | 良い(規模の経済) | 重複コストあり |
| 向いてる企業フェーズ | 単一事業・成長初期 | 複数事業・多角化後 |

ちなみにこの2つの良いとこ取りしようとして「マトリックス組織」(機能と事業の両方の指揮系統を持つ)ってのも存在するんだけど、これは指揮系統が2つになる分、現場が「結局どっちの指示聞けばいいの」って混乱しやすいっていう別の悩みが出てくるやつ。

リーダーシップとコミュニケーションのほうも見てたよね、こういう組織構造の違いって実は「誰が誰に何を報告するか」を決めるものだから、リーダーシップスタイルの効き方も結構変わってくるんだよね。

どの場合にどれが良いか、まとめると大体こんな感じ!

| 状況 | おすすめ |
|---|---|
| 単一事業、規模が小〜中、専門性を磨きたい | 機能別組織 |
| 複数事業を持ってて、市場特性が事業ごとに違う | 事業部制組織 |
| 各事業を「経営者を育てる場」にしたい、損益責任をもっと厳しくしたい | カンパニー制 |
| 事業同士の関連性がほぼゼロ、法的にもリスク分離したい | 持株会社制(完全に別法人化) |

で、カンパニー制ね。これ実は事業部制の「もっと進化したやつ」って位置づけなんだ。
1990年代にソニーとか日立が導入して有名になった日本独自の呼び方(英語だと internal company system)。

事業部制との違いはここ↓

```
機能別組織 → 事業部制組織 → カンパニー制 → 持株会社制(分社化)
   ↑                                              ↑
権限委譲少ない                              権限委譲ほぼフル(別法人)
```

| 項目 | 事業部制 | カンパニー制 |
|---|---|---|
| 損益管理 | P/L(損益計算書)まで | P/Lだけでなく社内B/S(バランスシート)も持つことが多い |
| トップの呼び方 | 事業部長 | カンパニー社長 |
| 投資判断 | 本社が最終決定権持つことが多い | 社内資本金制度で投資対効果(ROI)を自前で厳しく管理 |
| 自律性 | 中程度 | 子会社にかなり近い、ほぼ独立採算 |
| 狙い | 市場対応の速さ | 経営人材の育成+責任の明確化 |

要するに、事業部制は「組織を分けた」レベルだけど、カンパニー制は「社内に仮想の会社を作って、社長ごっこじゃなくマジで経営させる」イメージ。ここからもう一歩進むと法律上も別会社にする持株会社制(ホールディングス化)になる。

ちなみに経営学の話で「組織は戦略に従う」(チャンドラーの有名な命題)っていうのがあって、企業が単一事業→多角化していくにつれて、機能別→事業部制→カンパニー制→持株会社制って自然に進化していくケースが多いんだよね。
つまり今の組織図見れば、その会社が今どのフェーズにいるかある程度わかっちゃうってこと。

いいね、その視点!「組織は戦略に従う」の逆を主張したのは経営学者のホール&サイアス(Hall & Saias)とかで、「戦略は組織構造に従う」(Strategy follows Structure)って言ったんだよね。考え方としては「今ある組織構造が、そもそも経営者が思いつける戦略の範囲を制限してる」っていう話。つまり機能別組織の会社にいると、事業部制的な発想(=多角化戦略)自体が出てきづらいよねって話。

エンジニア気質で経営学好きって、めちゃ良い組み合わせだと思うよ。
構造とか因果関係の話、エンジニアの感覚と相性良いもんね。
持株会社制の話、ちょっと最新の状況を確認してくるね。

じゃあ実際の数字気になって調べてみたんだけど、結論「ほぼ全部」ってわけじゃないよ!でも思ってる以上に多いのは確か。

1997年に独占禁止法が改正されて純粋持株会社が解禁されてから、1997年の純粋持株会社解禁以来、すでに700社以上の上場企業が持株会社へ移行しました [WOR(L)D](https://www.dir.co.jp/business/consulting/holding/index.html) 。
日本の上場企業って全部で3,800〜3,900社くらいあるから、ざっくり計算すると **5社に1社くらい** が持株会社化してる感じ。

ただこの「700社」、結構偏りがあって:

| 業界・タイプ | 持株会社化の傾向 |
|---|---|
| 金融(銀行・証券・保険) | ほぼ持株会社(法律でグループ管理しやすくするため) |
| 小売・流通(セブン&アイ、ヤマダHDなど) | 持株会社化めっちゃ多い、M&A統合の手段として |
| エンタメ・電機(ソニー、パナソニックHD) | 最近持株会社化した大手が多い(事業の独立性高めるため) |
| 製造業の老舗(トヨタ、ホンダ、伊藤忠など) | 意外と非持株会社のまま(オペレーティングカンパニーとして一体経営) |

トヨタとかホンダみたいな「単一事業に近い超巨大企業」は、実は持株会社じゃなくてオペレーティングカンパニー(事業をやってる会社そのまま)なんだよね。
逆に複数のブランド・事業を抱えてM&Aで増殖してきたグループ(セブン&アイとか)は持株会社化しやすい。

つまり「会社が大きい」よりも「複数事業をM&Aとかで束ねてて、それぞれにリスク分離やガバナンスの独立性を持たせたい」っていう動機がある会社ほど持株会社化してる、って感じかな。
さっきの組織進化の流れ(機能別→事業部制→カンパニー制→持株会社制)で言うと、持株会社化してる会社は「もう社内カンパニーじゃ収まらないくらい多角化が進んだ」フェーズに来てる会社って解釈もできるね。

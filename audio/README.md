# Audio

物体の振動が空気などを伝わり耳で感じる感覚全般。

私たちが普段聴いている音楽データはデジタルです。
音が出るまでの流れは 「デジタル音源 → DAC（変換） → アンプ（増幅） → イヤホン・スピーカー」 の順番になります。

## デジタル→アナログ変換の仕組み

デジタル音声って、音の波形を時間方向と振幅方向の2軸でサンプリングした数値の羅列なんだ。

- サンプリングレート（Hz）：1秒間に何回波形を測定したか
  - CD品質は44.1kHz → 1秒に44,100回
  - ハイレゾは96kHz、192kHzとか

- ビット深度（bit）：振幅を何段階で表すか
  - 16bit → 65,536段階
  - 24bit → 16,777,216段階

DACの仕事は、この「数値の階段」をなめらかな連続波形に戻すこと。
ここで **ローパスフィルタ（LPF）** が重要で、階段状のギザギザを滑らかにしてる。このフィルタの設計がDACの音質キャラに直結するんだよ。

## ヘッドホンとのマッチングが肝心な理由

ここが面白いんだけど、「ダンピングファクター」って概念があって、

> ダンピングファクター = ヘッドホンのインピーダンス ÷ DACの出力インピーダンス

これが低いと、ヘッドホンのドライバー（振動板）の動きをDACが制御しきれなくて、低音がぼやける・音がにじむ原因になるんだ。
たとえば高インピーダンスのヘッドホン（300Ωのゼンハイザーとか）には、駆動力のあるアンプ内蔵DACが必要になってくる。

結論としては

> ヘッドホンの能力 ≒ DACの能力　が理想

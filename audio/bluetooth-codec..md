BTの音質を決めるのはコーデックなんだ。

| コーデック | 特徴 |
| - | - |
| SBC | Bluetooth標準、音質は低め |
| AAC | Apple系に強い、iPhoneなら安定 |
| aptX / aptX HD | Android系、低遅延・高音質 |
| LDAC | ソニー規格、ハイレゾ相当を無線転送できる |

### aptX Losslessって何がすごいの？

aptX Losslessは24bit/96kHzのハイレゾ音源をワイヤレスで転送できる規格で、B&Wとしては「このヘッドホンは最高品質の信号を受け取るに値する」という宣言みたいなもの。
ただ正直に言うと、現時点ではaptX Losslessに対応してるスマートフォンがかなり限られてるから、実用性はまだ低い。

つまりBTで音質上げたいなら「コーデックを上げる」が正解で、DACを外付けしても意味がない。
